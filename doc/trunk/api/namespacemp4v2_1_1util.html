Detailed Description

(private) Command-line utility support.

WARNING: THIS IS A PRIVATE NAMESPACE. NOT FOR PUBLIC CONSUMPTION.

This namespace is used for command-line utilities. Some symbols from this namespace are exported from libmp4v2 in order to support new functionality which may or may not make it into some form of public API, at which time it will be moved out of this namespace.

Copy-out and close an mp4 file.

This function writes out a copy of all the atoms from an open mp4 file. The new file is written out in optimized-form, so a subsequent optimize operation is not necessary. Both files are then closed.

Parameters:
hFile handle of file to close.
copyFileName name of new file to copy to.
Returns:
true on success, false on failure.
