MP4TrackId  trackId,
MP4Duration  duration,

)

Convert duration from track time scale to an arbitrary time scale.

MP4ConvertFromTrackDuration converts a duration such as a sample duration from the track time scale to another time scale. This can be used by a player application to map all track samples to a common time scale.

Parameters:
hFile handle of file for operation.
trackId id of track for operation.
duration value to be converted.
timeScale time scale in ticks per second.
Returns:
On success, the duration in arbitrary time scale units. On error, 0.
