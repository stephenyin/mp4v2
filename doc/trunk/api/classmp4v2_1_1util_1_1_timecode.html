Detailed Description

Class representing SMPTE timecode.

Objects of this class represent a specific time or duration and can be converted to/from string values, and various epochs.

The standard pattern for string representation is as follows:

  • HH:MM:SS:FF
  • HH:MM:SS.DDD
where:
  • HH specifies hours
  • MM specifies minutes
  • SS specifies seconds
  • : specifies normal timecode
  • FF specifies the frame number
  • . specifies decimal fractions of a second follow
  • DDD specifies decimal fractions of a second, rounded down to closest scale
