Detailed Description

Utility general program class.

This class provides a base implementation for a general utility which helps meet behavioral criteria for command-line executables.

Inherit batch processing ability is also provided and is used optionally by the concrete implementation.

Criteria and guidelines for utility behavior in MP4v2 are as follows:

  • exit with 0 when the utility succeeds at its main task, 1 for failure.
  • print brief-usage when no arguments are supplied and exit with 1.
  • print brief-usage when too many arguments are supplied and exit with 1.
  • issue brief-usage when invalid argument is supplied and exit with 1.
  • support --help option and exit with 0.
  • support --version option and exit with 0.
  • utilities which create new output files should never over-write an existing file unless given the project's universal '-o' or '--overwrite' option.
